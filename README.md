# Klasifikasi_resistensi_antibiotik_RF
Pengembangan model klasifikasi resistensi antibiotik berbasis data EHR dengan menggunakan algoritma Random Forest 

penelitian membandingkan 3 metode dalam penyeimbangan data imbalance di algoritma RF yaitu menggunakan Classweight dan SMOTE
dengan hasil pada validasi akhir menunjukkan SMOTE mendapat nilai akurasi 77% dan CLass weight 76%
